### 1. Python 基础知识

#### 变量和数据类型

1. **Python 的数据类型有哪些？**
    - 数字：`int`，`float`，`complex`
    - 字符串：`str`
    - 列表：`list`
    - 元组：`tuple`
    - 集合：`set`
    - 字典：`dict`
    - 布尔值：`bool`
    - None 类型：`NoneType`

2. **什么是可变和不可变类型？**
    - 可变类型：`list`，`set`，`dict`（可以修改内容）
    - 不可变类型：`int`，`float`，`tuple`，`str`，`frozenset`，`bytes`（不可以修改内容）

3. **如何在 Python 中进行类型转换？**
    ```python
    int("123")        # 转为整数
    float("3.14")     # 转为浮点数
    str(123)          # 转为字符串
    list((1, 2, 3))   # 转为列表
    ```

#### 控制流

1. **if 语句的用法**
    ```python
    if x > 0:
        print("正数")

        print("负数")
    else:
        print("零")
    ```

2. **for 循环和 while 循环**
    ```python
    for i in range(5):  # 0到4
    ```

3. **break 和 continue**
    ```python

        if i == 2:
    ```

### 2. 高级特性

#### 装饰器

装饰器是 Python 中的一个重要概念，允许你在函数调用前后添加额外的功能。

```python
def decorator(func):
    def wrapper():
        print("Before function call")
        func()
        print("After function call")
    return wrapper

@decorator
def greet():
    print("Hello!")

greet()  # 调用时会输出：Before function call -> Hello -> After function call
```

#### 生成器和迭代器

生成器使用 `yield` 来生成一个迭代器。它节省内存，适用于处理大数据。

```python
def count_up_to(max):
    count = 1
    while count <= max:
        yield count
        count += 1

counter = count_up_to(5)
for num in counter:
    print(num)
```

#### 上下文管理器

上下文管理器用于管理资源的获取和释放，通常使用 `with` 语句。

```python
class MyContextManager:
    def __enter__(self):
        print("Enter")
        return self

    def __exit__(self, exc_type, exc_value, traceback):
        print("Exit")

with MyContextManager():
    print("Inside")
```

#### 闭包和匿名函数

闭包是指函数返回一个函数，返回的函数可以访问定义时的变量。

```python
def outer(x):
    def inner(y):
        return x + y
    return inner

closure = outer(10)
print(closure(5))  # 输出 15
```

#### 常见数据结构

- **数组（List）**：支持按索引存取。
- **栈（Stack）**：先进后出（LIFO）。

- **哈希表（Hash Table）**：基于键值对存储。
- **集合（Set）**：无重复的元素集合。

#### 排序与查找

- **快速排序**：分治法，将大问题分解成小问题。
- **二分查找**：用于有序列表的查找。

#### 动态规划

动态规划是一种优化递归算法的方法，将中间结果存储起来避免重复计算。

json 和 pickle
json：用于处理 JSON 数据，可以将 Python 对象转换为 JSON 字符串，也可以从 JSON 字符串中解析数据。

import json

# 将字典转为 JSON 字符串
data = {"name": "Alice", "age": 25}
json_str = json.dumps(data)
print(json_str)  # 输出：{"name": "Alice", "age": 25}

# 将 JSON 字符串转为字典
data = json.loads(json_str)
print(data)  # 输出：{'name': 'Alice', 'age': 25}
pickle：用于序列化和反序列化 Python 对象，通常用于存储和传输 Python 特有的对象。

import pickle

# 序列化对象
data = {"name": "Alice", "age": 25}
with open("data.pickle", "wb") as f:
    pickle.dump(data, f)

# 反序列化对象
with open("data.pickle", "rb") as f:
    data = pickle.load(f)
print(data)  # 输出：{'name': 'Alice', 'age': 25}

Flask / Django 基础

两数之和
给定一个数组和一个目标值，找到数组中两个数的和等于目标值的索引。

def two_sum(nums, target):
    seen = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i

class Stack:
    def __init__(self):
        self.stack = []

    def push(self, item):
        self.stack.append(item)

    def pop(self):
        return self.stack.pop() if self.stack else None
链表反转
给定一个链表，反转它。

class ListNode:
    def __init__(self, value=0, next=None):
        self.value = value
        self.next = next

def reverse_linked_list(head):
    prev = None
    while head:
        next_node = head.next
        head.next = prev
        prev = head
        head = next_node
    return prev
